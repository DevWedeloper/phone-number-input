## Overview

This page describes how to use the phone number input component in an **Angular v19+** project.  
The integration targets **modern Angular applications** using **standalone components**.

## Requirements

- Angular **v19 or higher**

## Installation

First, install the required Maskito packages and phone number parsing library:

<Code lang="shell" code="pnpm add @maskito/core @maskito/kit @maskito/phone libphonenumber-js" />

Next, install the Angular wrapper package for phone number input:

<Code lang="shell" code="pnpm add ngx-phone-number-input" />
